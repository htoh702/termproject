git 설치 이후
-> git config --global user.name 유저이름	// 유저이름 설정
-> git config --global user.email 유저메일	// 유저이메일 설정
-> 원하는 위치에 github에서 내려받을 파일 생성
-> cd c:/위치	// 파일 생성 후 그 파일로 이동
-> git clone github주소	// github에서 파일 내려받기
-> cd c:/파일/프로젝트	// 내려받은 프로젝트로 들어가기
-> 거기에 작업한 파일을 저장
-> git add 파일	// git에 작업한 파일 저장
-> git commit -m
